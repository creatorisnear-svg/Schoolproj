Commands

Every slash command available in RolePlayManager

Staff & Administration
/addstaff Add a user or role as staff
/removestaff Remove a user or role from staff
/stafflist View the current staff list
/setlogchannel Set the channel for bot logs
/enablecommands Enable or disable command categories
/reloadconfig Reload the server configuration
Verification & Security
/verifysystemsetup Set up member verification
/verify Manually verify a member
/antipromotingsetup Configure the invite link filter
/welcomesystemsetup Set up welcome messages
Strikes & Discipline
/strikesystemsetup Configure the strike system
/strike Issue a strike to a member
/removestrike Remove a strike from a member
Priority Tracker
/prioritytrackersetup Configure the priority system
/activepriority Start a priority session
/deactivatepriority End a priority session
/prioritycooldown Set cooldown between priorities
/priorityrequest Request a priority (staff)
Roleplay & Databases
/roleplaycommandsetup Set up RP commands like /911
/civiliandatabase Access the civilian CAD database
/leodatabase Access the LEO CAD database
/firedepartmentdatabase Access the fire department database
/setrp Set the current roleplay status
/unsetrp Clear the current roleplay status
/roleplaycalendersetup Set up the RP event calendar
Tickets & Roles
/ticketsupportsetup Set up the ticket support system
/rolerequestadd Add roles to the role request menu
/rolerequest Open the role request menu
/manageroles Manage a member's roles
/reactionrolemessage Create a reaction role message
Utility
/sticky Create an auto-reposting sticky message
/stickylist List all active sticky messages
/clear Bulk delete messages in a channel
/embed Create a custom embed message
/help View all available commands
/invite Get the bot invite link
